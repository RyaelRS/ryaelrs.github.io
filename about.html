About this project

I was inspired to make this website because during studying my Social Studies Pace I learned about The American Civil War. I was very interested in it and made a website about it for people to learn more about this War with the help of my website and to understand the reason behind the conflict between the States.

How this site was made?

This webisite was made with a template in startbootstrap.com. Bootstrap is an existing CSS libary that already has styling that is responsive depending on the user's browser screen size, the bootstrap theme I used already included a sample page as my starting point. I used Visual Studio Code to write HTML and CSS.

Future improvements?

It would be good to have a search bar so people can find specific words.
